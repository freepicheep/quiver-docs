# Documentation Site for [Quiver](https://github.com/freepicheep/quiver?tab=readme-ov-file)

Quiver is an extremely fast Nu package and project manager, written in Rust. If you live in the terminal, try using Nushell as your main shell for a while. If you work with data, you will enjoy using Nu instead of languages like Python for data manipulation. Quiver helps you build projects and share them, just like you would use `uv` or `pip` for python.
